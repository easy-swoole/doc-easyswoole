# 入门教程

## 安装

### 第一节 安装swoole 约 5分钟
1. 使用一台腾讯云 Cenos7.4 机器 已经装好了LNMP 大概阐述下当前环境
2. 通过github访问swoole项目仓库  然后选择最新4.3 or 4.4  下载
3. 利用yum 安装swoole相关依赖
4. 编译安装swoole 并且启用 openssl socket 以及指定php config路径
5. 添加swoole.so到php.ini 并 使用php --ri swoole 检查
